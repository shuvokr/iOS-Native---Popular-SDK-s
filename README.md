### Installation

##### HTTP

```sh
$ cd "directory address"
$ git clone https://github.com/shuvokr/iOS-Native---Popular-SDK-s.git
$ cd iOS-Native---Popular-SDK-s 
$ pod install
```

##### Github CLI

```sh
$ cd "directory address"
$ gh repo clone shuvokr/iOS-Native---Popular-SDK-s
```
